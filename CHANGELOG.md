# TurboTransfer 更新日志

## v1.1.0 (2026-06-30)

### 新增功能

- **连接地址展示** — 侧边栏底部展示本机所有局域网连接地址，其他设备可直接访问；点击地址一键复制到剪贴板
- **系统托盘后台运行** — 点击关闭按钮最小化到系统托盘而非退出程序；托盘右键菜单支持"显示窗口"和"退出程序"；双击托盘图标恢复窗口
- **已连接设备追踪** — 新增 `GET /api/clients` 接口，记录所有访问过本机服务的客户端 IP、浏览器、设备型号、请求次数和最后活跃时间；设备页面新增"已连接设备"展示区域
- **Client Hints 设备识别** — 响应头下发 `Accept-CH`，Chrome 浏览器后续请求会携带设备型号（`Sec-CH-UA-Model`）和平台版本，用于更精确的设备识别
- **文件库排序** — 支持按名称、大小、时间排序，正序倒序可选
- **torrent 文件类型识别** — `.torrent` 文件显示专属图标和 emerald 绿色主题标签

### 界面改版

- **暗色科技风主题** — 全新深色配色方案（深蓝底色 + 蓝色辉光 accent），面板采用半透明玻璃质感，交互元素带辉光阴影反馈
- **应用图标** — 新增 256px 应用图标和 32px 托盘图标

### 优化改进

- **端口变更** — 服务端口从 `8080` 改为 `9527`，降低与其他服务冲突概率
- **系统文件过滤** — 文件库自动过滤 `desktop.ini`、`Thumbs.db`、`.DS_Store` 等系统文件
- **请求计数修正** — 仅统计 `/api/*` 和 `/upload/chunk` 的有效 API 请求，避免静态资源加载导致计数虚高
- **文件时间戳** — `GET /api/files` 返回值新增 `lastModified` 字段，支持按时间排序

## v1.0.0 (2026-06-30)

### 新增功能

- **Electron 桌面打包** — 重构打包流程为 Electron Builder NSIS 安装包，替换原 jpackage/WiX 方案；支持自定义安装目录、桌面快捷方式、开始菜单
- **内嵌 JRE** — 打包时通过 `jlink` 生成精简 JRE，用户无需预装 Java 环境
- **自定义传输目录** — 设置页面支持通过原生目录对话框选择文件传输存储路径

### 问题修复

- **Maven 构建修复** — 修复 `build-dist.bat` 中 `call "mvn.cmd"` 带引号导致 cmd.exe 静默失败的问题，改用 `where` 获取 Maven 完整路径
